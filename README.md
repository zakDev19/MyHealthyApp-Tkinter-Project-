# MyHealthyApp

Application graphique développée en Python avec Tkinter permettant d'organiser la répartition des participants pour un repas partagé.

## Description

MyHealthyApp aide à équilibrer les contributions des participants entre plusieurs catégories alimentaires :

- Entrée
- Plat
- Dessert
- Boisson

L'application calcule automatiquement le nombre maximum de participants par catégorie afin d'assurer une répartition équilibrée.

## Fonctionnalités

- Saisie du nombre total de participants
- Ajout de participants avec prénom et nom
- Attribution à une catégorie alimentaire
- Vérification des doublons
- Contrôle des catégories complètes
- Affichage des participants enregistrés
- Jauges circulaires de progression
- Barres de progression par catégorie
- Résumé automatique des éléments manquants

## Technologies utilisées

- Python 3
- Tkinter



## Objectif du projet

Projet réalisé dans le cadre d'un cours d'Ingénierie des Systèmes afin de mettre en pratique :

- La programmation événementielle
- Les interfaces graphiques avec Tkinter
- La gestion de données utilisateurs
- La visualisation de données en temps réel
